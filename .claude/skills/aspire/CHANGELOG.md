# Changelog: aspire

## 1.0.0 — 2026-03-23

- Initial release — Aspire CLI 13.2+ command reference
- Covers: start, stop, describe, wait, logs, otel logs/traces/spans, resource management, docs search, MCP tools
- Key workflows: running in agent environments, debugging issues, adding integrations
- Important rules: use `aspire start` (not `aspire run`), `--isolated` for worktrees, never install workload
